<h1 dir="rtl">چگونه می توان شمارنده را در پایتون پیاده سازی کرد؟</h1>
<br>
<p dir="rtl">تا قبل از منتشر شدن نسخه پایتون ۳.۴ ، زبان پایتون پیاده سازی برای شمارنده ارائه نکرده بود. برنامه نویسان برای استفاده از شمارنده یا خود آن ها با توجه به نیازی که داشتند، پیاده سازی خاصی از شمارنده را انجام می دادند و از آن استفاده می کردند یا از کتابخانه های آماده که بقیه برنامه نویسان آن ها را آماده و عرضه کرده بودند استفاده می کردند. از نسخه پایتون ۳.۴ شمارنده به پایتون اضافه شد و سپس به نسخه های قبلی پایتون نیز اضافه شد ولی ما برای بهره بردن از امکانات بیشتر همچنان چگونگی استفاده از شمارنده و پیاده سازی دستی آن مورد توجه است که ما به بررسی چند مورد از آن ها می پردازیم.</p>

<p dir="rtl">یک روش استفاده از شمارنده استفاده از کتابخانه های آماده است که دیگر برنامه نویسان آن‌ها را آماده کرده‌اند که قابلیت های بیشتری دارند. می توان به کتابخانه های aenum و enum34 اشاره کرد. enum34 نسبت به دو نسخه پایتون ۳ و پایتون ۲ سازگار نیست برای همین استفاده از aenum توصیه می شود. برای نصب این دو کتابخانه به روش زیر عمب می کنیم: </p>

<ul>
<li>$ pip install aenum</li>
<li>$ pip install enum34</li>
</ul>
